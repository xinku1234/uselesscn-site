## Safe submission notes

- Do not mass-submit the same wording everywhere.
- For Reddit, read subreddit rules first and disclose that the site is yours.
- For Product Hunt / Indie Hackers / HN, use the creator’s own account, not an automated bot account.
- Avoid AI-tool directories unless the directory accepts general web tools; UselessCN is not an AI product.
- Prefer a few relevant communities over dozens of low-quality directory submissions.
